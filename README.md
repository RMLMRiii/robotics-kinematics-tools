# Robotics Kinematics Tools

A Python toolkit for basic robotics kinematics calculations, including forward position and workspace-style analysis.

## Features
- Computes 2D robotic arm end-effector position
- Uses joint angles and link lengths
- Organized function-based structure
- Includes sample robot configuration data
- Separate configuration viewer script

## Technologies Used
- Python
- CSV file handling
- Trigonometry
- Basic robotics kinematics

## How to Run
Run the script in Python:

python forward_kinematics.py

To view the sample robot configuration data:

python config_viewer.py
